CPI API Documentation

Here you will find information on consuming the cpi api.



ADDING CPI Data

POST /api/v1/cpi - Add some new news to our database!

GETTING CPI Data

GET /api/v1/cpi - Get all the cpi data from our database!
GET /api/v1/cpi/<id> - Get a specific cpi data point from our database!

DELETING CPI Data

DELETE /api/v1/cpi/<id> - Get all the cpi from our database!
